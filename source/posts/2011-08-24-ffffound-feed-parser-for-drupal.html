I'm really happy to announce my first contributed Drupal module. Feeds: FFFFOUND! parser provides a parser for the feeds from FFFFOUND! with support for the following data:

  • Feed title
  • GUID
  • FFFFOUND! URL
  • Image title
  • Description
  • Author
  • Original image URL
  • Thumbnail URL
  • Source image URL
  • Referrer URL
  • Saved by count
  • Published on (Datetime)
  • Published on (Timestamp)

You can also, if used with File Field Sources, use it to backup all the images you've saved on FFFFOUND! to your own server.
