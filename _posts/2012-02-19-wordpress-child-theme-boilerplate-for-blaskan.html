To make things even easier for anyone who wants to create a child theme for Blaskan I've created a Blaskan child theme boilerplate. Download it from GitHub and follow the instructions in readme.txt.

  • Download
  • Source code

A child theme should always be used when you need to modify or extend the functionality of a theme. If you edit any of the files that comes with Blaskan your changes will be overwritten each time you upgrade to a newer version.
