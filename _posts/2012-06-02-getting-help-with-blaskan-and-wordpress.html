Do you need help using, modifying or extending Blaskan? Unfortunately I don't have the time to provide personal support via email, Twitter and so on but there are many ways to get help.

First off, always use a child theme if you need to make any changes to the code. Learn more about the starterkit I've made for you.

Get help from the WordPress community

  • Post your question in the forums on WordPress.org, WordPress Answers or WP Questions. Blaskan also has it's own forum on WordPress.org.
  • Read the online manual: WordPress Codex. A good place to start is the page New to WordPress – Where to Start, especially the Need More Help part.
  • If you're hosting your blog on WordPress.com take a look at their support pages and forums.

I'm glad you're using WordPress and Blaskan – now take advantage of the power of the community.

If you find any bugs please report them on GitHub or directly via email.
